# Recruitment

## Purpose
Recruitment pipeline — job postings and sales pipeline view. Manages the hiring workflow from posting creation through candidate tracking.

## File Inventory
| File | Role |
|------|------|
| RecruitmentPage.jsx | Main recruitment page — job posting list and pipeline overview |
| RecruitmentPage.css | Styles for recruitment page layout and pipeline columns |
| SalesPipelinePage.jsx | Sales-style pipeline view for tracking candidates through stages |
| SalesPipelinePage.css | Styles for pipeline kanban layout and stage cards |

## Dependencies
- Stores: people store (recruitment pipeline, candidates, job postings)
- Data: `src/data/config/recruitment-config.js`
